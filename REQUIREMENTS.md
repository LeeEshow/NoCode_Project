# 個人理財雲端系統 — 需求想法文件

> 版本：1.0（2026-04-22）

## 一、專案概述

建立一套個人理財管理的雲端 Web 系統，主要功能為台股投資組合追蹤、買賣紀錄管理、即時股價整合、長期投報計畫試算，以及每日收盤後自動快照記錄與報酬率走勢追蹤。

**核心使用場景：**
- 查看目前持股狀況與即時損益
- 記錄買賣交易，系統自動計算成本與均價
- 追蹤長期複利目標與實際達成對比
- 管理外幣與債券資產，換算台幣合計
- 每日自動快照（台灣時間 14:00，台股收盤後 30 分鐘），長期觀察投資績效走勢

## 二、成本計算邏輯規格

### 買入時（兩種方法相同）
```
新均價 = (原總成本 + 本次買入金額) ÷ (原持有股數 + 本次買入股數)
本次買入金額 = 股數 × 單價 + 手續費
```

### 賣出時

**方法一：獲利歸還法**
```
剩餘總成本 = 原總成本 - 本次賣出收益
本次賣出收益 = 股數 × 成交價 - 手續費
剩餘均價 = 剩餘總成本 ÷ 剩餘股數
已實現損益（獨立記錄）= 賣出收益 - 賣出成本比例
```

**方法二：成本保留法（預設）**
```
剩餘總成本 = 原總成本（不變）
剩餘均價 = 原總成本 ÷ 剩餘股數
已實現損益 = 本次賣出收益（整筆計入）
```

### 範例驗算（00894）
- 持有：2張，均價 31.7元，總成本 63,400元
- 賣出：0.5張，成交價 40.5元，收益 20,250元
- 方法一：剩餘均價 = (63,400 - 20,250) ÷ 1,500 = **28.77元**
- 方法二：剩餘均價 = 63,400 ÷ 1,500 = **42.27元**

> **成本計算架構**：後端僅儲存每筆交易原始紀錄，前端 ViewModel 依全部交易紀錄動態運算均價與總成本，計算完成後呼叫 `POST /holdings/recalculate` 將結果寫回後端。
> **成本方法切換**：使用者於設定頁手動點擊「重新計算」按鈕觸發，系統依當前設定的方法重新計算所有持股成本。

## 三、投報計畫數學模型（MARC Model）

> **模型名稱**：Macro-Adjusted Return Compound Model（宏觀調整型複利模型）
> **設計來源**：Global Macro Strategist 建議，採三層報酬率疊加架構。

### 核心公式

```
名目有效報酬率  r_nominal = r_base × k_risk
實質報酬率      r_real    = r_nominal - (pi_base + pi_shock)
```

### 參數定義與預設值

| 參數 | 說明 | 預設值 | 可調範圍 |
|------|------|--------|---------|
| `r_base` | 基礎殖利率（台股長期年化均值） | `8%` | 5%–15% |
| `pi_base` | 長期通膨基準（台灣 CPI 長期均值） | `2.0%` | — |
| `pi_shock` | 通膨情境加成（由情境切換控制） | `0%` | — |
| `k_risk` | 景氣風險係數（對應景氣燈號） | `1.00` | 見下表 |
| `plan_years` | 計畫年數（固定） | `30` | 固定 |

### 通膨情境切換

| 情境 | `pi_base` | `pi_shock` | 綜合通膨率 |
|------|----------|------------|----------|
| 🟢 低通膨 | `1.5%` | `0%` | `1.5%` |
| 🟡 基準（預設） | `2.0%` | `0%` | `2.0%` |
| 🔴 高通膨 | `2.0%` | `1.5%` | `3.5%` |

### 景氣風險係數對照表（k_risk）

| 景氣燈號 | `k_risk` | 邏輯依據 |
|---------|--------|---------|
| 🔴 紅燈（過熱，≥38分） | `0.85` | 均值回歸壓力，上方報酬有頂 |
| 🟡 黃紅燈（32–37分） | `0.95` | 接近過熱，溢酬收斂 |
| 🟢 綠燈（23–31分） | `1.00` | 正常景氣，全額計入（預設） |
| 🔵 黃藍燈（17–22分） | `1.05` | 底部醞釀，逢低布局溢酬 |
| 🔵 藍燈（衰退，<17分） | `1.10` | 長線折價最大，期望值反升 |

### 逐年試算邏輯（前端 ViewModel 實作）

```typescript
// buildMARCRows() — 前端 models/planModel.ts 實作
for n = 1 to plan_years:
  totalNominal    = (totalNominal + annualInvest) × (1 + r_nominal)
  capitalInvested = capitalInvested + annualInvest
  inflationFactor = inflationFactor × (1 + pi_total)
  totalReal       = totalNominal / inflationFactor
  interest        = totalNominal - capitalInvested
```

| 欄位 | 說明 | 顯示 |
|------|------|------|
| 年次（N） | 第 N 年 | 主欄 |
| 西元年 | 起始年 + N - 1 | 主欄 |
| 累積投入 | 純本金累積 | 主欄 |
| 當年利息 | 名目利息收益 | 主欄 |
| 名目總資產 | 未扣通膨 | **主顯示** |
| 實質總資產 | 購買力折現 | 灰字輔助 |
| 名目報酬率 | `r_base × k_risk` | tooltip |
| 實質報酬率 | 名目 - 通膨 | tooltip |

- 第 10 / 15 / 20 / 30 年高亮里程碑標記
- 顯示 20 年 / 30 年名目 & 實質總資產摘要卡片
